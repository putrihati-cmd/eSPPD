# Document Templates

Place your DOCX templates here with the following placeholders:

## template_sppd.docx

Placeholders:

- `{{NOMOR_SPPD}}` - Nomor SPPD
- `{{NAMA}}` - Nama pegawai
- `{{NIP}}` - NIP pegawai
- `{{JABATAN}}` - Jabatan
- `{{PANGKAT}}` - Pangkat
- `{{GOLONGAN}}` - Golongan
- `{{UNIT_KERJA}}` - Unit kerja
- `{{TUJUAN}}` - Tujuan perjalanan
- `{{KEPERLUAN}}` - Keperluan
- `{{TANGGAL_BERANGKAT}}` - Tanggal berangkat
- `{{TANGGAL_KEMBALI}}` - Tanggal kembali
- `{{LAMA_PERJALANAN}}` - Lama perjalanan (hari)
- `{{SUMBER_DANA}}` - Sumber dana
- `{{PEJABAT_TTD}}` - Nama pejabat penandatangan
- `{{JABATAN_TTD}}` - Jabatan pejabat
- `{{NIP_TTD}}` - NIP pejabat
- `{{TANGGAL_SPPD}}` - Tanggal pembuatan

## template_laporan.docx

Placeholders:

- `{{NOMOR_LAPORAN}}` - Nomor laporan
- `{{NAMA}}` - Nama pegawai
- `{{NIP}}` - NIP pegawai
- `{{JABATAN}}` - Jabatan
- `{{NOMOR_SPPD}}` - Nomor SPPD terkait
- `{{TUJUAN}}` - Tujuan
- `{{TANGGAL_BERANGKAT}}` - Tanggal berangkat
- `{{TANGGAL_KEMBALI}}` - Tanggal kembali

- `{{SARAN}}` - Saran

Create your templates in Microsoft Word and use `{{PLACEHOLDER}}` syntax for dynamic values.
